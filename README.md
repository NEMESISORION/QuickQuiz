# ⚡ QuickQuiz

Fast. Simple. Effective Learning.

A full-stack web application that allows teachers to create quizzes and students to take them — with separate dashboards and role-based access control.

## 🚀 Live Demo
[quickquiz-production-d5d0.up.railway.app](https://quickquiz-production-d5d0.up.railway.app)

**Demo Credentials:**
- Admin: `admin` / `1234`
- Student: `student` / `1234`

## 🛠️ Built With
- PHP
- JavaScript
- HTML / CSS
- MySQL (Supabase)
- Deployed on Railway

## ✨ Features
- Role-based authentication (Admin / Student)
- Teachers can create and manage quizzes
- Students have a separate dashboard to take quizzes
- Clean, minimal UI

## 📦 Run Locally
1. Clone the repo
2. Import the SQL file into your local MySQL database
3. Update `config.php` with your database credentials
4. Run with XAMPP or any PHP server
